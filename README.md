# Dotfiles
Custom dotfiles I use on **Omarchy Linux**.

###  Waybar

![Waybar Preview](assets/waybar.png)
![Waybar Preview](assets/waybar2.png)
![Waybar Preview](assets/waybar4.png)


Custom Waybar setup for Hyprland, originally based on one of the styles from  [JaKooLit’s Dots](https://github.com/JaKooLit/Hyprland-Dots/tree/c862835221decf03b6640bb43e4044861cceaa04/config/waybar), which I further modified for better integration with my setup.

- **Workspaces** – Clean Hyprland icons with smooth transitions.
- **Audio** – PipeWire/ALSA volume widget with scroll actions using default omarchy scripts.
- **Visualiser** – Cava Visualiser with playerctl controls.
- **Backlight** – Brightness control with scroll actions default omarchy scripts.
- **VPN** – Auto-detects  and can toggle WireGuard/OpenVPN via custom scripts,based on [lajosdeme’s script](https://github.com/lajosdeme/proton-vpn-arch/blob/7a01b2d168e1e172a31840e25db5e802326dd5c6/proton-vpn-setup.sh).
